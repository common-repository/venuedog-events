A sidebar widget that displays events from VenueDog.com by date. Choose the categories to pull from and the number of days to display at a time.

== Description ==

Pull events from VenueDog.com and put them in the sidebar. Keeps your visits up-to-date on what's happening.

* [Project Homepage](http://venuedog.com)

== Installation ==

1. Upload plugin to the 'wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= When will VenueDog come to my city? =

VenueDog events can be added to your city by any user. Once a community is founded in a city, a full-time VeneueDog City Manager can be assigned to better facilitate your local events and venues.
